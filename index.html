What is Ionic?

  • mobile optimized library for HTML/JS/CSS
  • A full stack hybrid mobile platform
    • Built on top of Cordova
    • and AngularJS and SASS
  • "Bootstrap for mobile apps"

What is Cordova (PhoneGap?)



Platform for building native mobile applications using HTML, CSS, and Javascript

Installation

Prerequisites

  1. Install Node/NPM
  2. Install Ruby/Sass
  3. (optional)Install XCode (iOS)
  4. (optional)Android SDK(Android)
  5. (optional)Install Gulp

Installation

npm install -g cordova ionic

Install Cordova and Ionic Globally

Installation - Gulp

npm install -g gulp

Getting started

Creating a new project


ionic start {name} {templateName}
ionic start myBadassApp sidemenu
cd myBadassApp
  • Named Templates
    • tabs (default)
    • sidemenu
    • maps
    • blank
  • Github repo url
  • Codepen url
  • Local directory (relative or absolute paths)

BUILDING & EMULATING


ionic build ios //builds app for platform
ionic emulate ios //will load the built app in xcode / emulator
ionic serve //will load the app in browser

// The path for our ionicons font files, relative to the built CSS in www/css
